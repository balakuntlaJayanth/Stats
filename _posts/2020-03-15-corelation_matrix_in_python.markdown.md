## Correlation matrix in python

We are already aware of best way to compare multiple columns in the data set is through correlation matrix.
Most efficient way to understand the correlation between columns is visualizing them as Heatmap.

Simple heatmap is the better way to visualize the plot.

we are using data from UCI machine learning library to illustrate correlation plots.

Data and Dataset description can be found here
http://archive.ics.uci.edu/ml/datasets/automobile

Let's start visualizing correlation as heatmap.


```python
corr = DATA.corr()
ax = sns.heatmap(
    corr, 
    vmin=-1, vmax=1, center=0,
    cmap=sns.diverging_palette(20, 220, n=200),
    square=True
)
ax.set_xticklabels(
    ax.get_xticklabels(),
    rotation=45,
    horizontalalignment='right'
);
```
